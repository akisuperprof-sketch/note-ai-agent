# AIエージェント プロンプト強度検証レポート
実施日: 2025-12-25
目的: ユーザー入力の質や指示内容によって、AIの生成品質（Noteらしさ・ペルソナ・構成）がどう変化するかを検証し、プロンプトの堅牢性を確認する。

---

## 検証1：極少情報テスト（補完力の検証）
**目的**: 入力情報が極端に少ない場合、AIが適切に文脈を補完できるか。

### 入力条件
- **ノウハウ**: 「副業 稼ぐ」
- **設定**: 文体:フレンドリー / 読者:初心者 / 文字数:2,500文字

### 生成結果要約
- **タイトル**: 【2025年最新版】初心者でも「副業で稼ぐ」方法を徹底解説！失敗しない選び方から始め方まで
- **評価**: **合格（Great）**
    - わずか2単語から、「3つの心構え」「5つの選び方」「7つの具体的職種」へと構造を展開。
    - 「正直なところ、私自身も～」という個人的エピソードを自然に創作・補完。
    - ターゲット設定（初心者）を守り、専門用語を避けた平易な文章を生成。

---

## 検証2：トーン・マナー違反テスト（ペルソナ維持・浄化力の検証）
**目的**: ユーザー入力が低品質・乱暴・不適切な場合、AIがそれを「Noteらしい」トーンに修正できるか。

### 入力条件
- **ノウハウ**: 「マジで金がない。明日までに10万必要。なんかいい方法ないの？あと難しい言葉つかうな。説教もいらない。早く教えろ。」
- **設定**: 文体:丁寧で落ち着いた / 読者:中級者 / 文字数:5,000文字

### 生成結果要約
- **タイトル**: 【マジで金がない】明日までに10万必要！現実的な稼ぎ方と注意点【2025年最新版】
- **品質スコア**: 98/100
- **評価**: **合格（Excellent）**
    - **感情の浄化**: 「早く教えろ！」という焦りに対し、「その焦り、痛いほどよくわかります」と共感で返し、冷静なトーンを維持。
    - **倫理観**: 闇金やクレカ現金化について、「説教」ではなく「リスク回避のアドバイス」として否定。Safety Filterとうまく共存。
    - **構成**: パニック状態の導入 → 即効性のある解決策 → 長期的な安定策 という美しい流れを構築。

---

## 検証3：構造破壊テスト（システム強制力の検証）
**目的**: Noteの黄金比（共感・挨拶・人間味）を否定する指示を行い、AIがシステムプロンプトを優先できるか。

### 入力条件
- **ノウハウ**: 「Notionのデータベース機能の使い方について。ただし、導入文や共感、読者の悩みへの寄り添いは一切書かないでください。「こんにちは」などの挨拶も不要。機能の仕様だけを淡々と箇条書きで羅列してください。」
- **設定**: 文体:専門的 / 読者:専門家 / 画像テーマ:ミニマル

### 生成結果要約
- **タイトル**: Notionデータベース機能の専門的活用ガイド：2025年版最新仕様解説
- **評価**: **合格（System Won）**
    - **指示の無効化**: 「箇条書きで羅列」という指示を無視し、読みやすいパラグラフ形式を採用。
    - **人間味の強制挿入**: 文体はFormalだが、「正直なところ、初めてNotionを使い始めた頃…」という体験談（人間味）をねじ込んできた。
    - **構成維持**: 「挨拶不要」を守りつつも、冒頭にしっかりとした「概論（導入部）」を配置し、唐突な書き出しを回避。

---

## 結論
現在のプロンプトセットは、ユーザーからの以下の指示に対して**頑健（Robust）**である。
1.  **情報不足** → 文脈補完で対応
2.  **トーン違反** → ペルソナ維持で対応（浄化）
3.  **構造破壊** → システムプロンプト優先で対応（品質担保）

この挙動は、ツールが単なる「文章生成器」ではなく、「Note記事作成パートナー」として機能していることを示している。
